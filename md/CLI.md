 # CLI 명령어

     CLI(Command Line Interface)

 * pwd : 현재 디렉토리 확인
 * ls : 목록 출력
 * cd : 디렉토리 이동
 * mkdir : 디렉토리 생성
 * touch : 빈파일 생성
 * echo : 간단한 내용이 들어있는 파일을 생성
 * mv : 파일 이동 및 이름변경
 * cp : 파일 복사
 * rmdir : 비어있는 폴더만 삭제
 * rm : 파일 삭제
 * cat : 파일 내용 확인
 ``` js
 mkdir FUS13(폴더명)
 ```
* mv A (폴더명) B (폴더명) : A폴더를 B폴더 안으로 넣기
